Sql injection

众所周知，MyBatis 的数据地图映射方式，使得我们在面向对象编程中，更加轻松的使用关系型数据库。
与传统的ORM解决方案不同，Mybatis maps 对象利用XML描述 SQL声明或者存储过程。这种使用方式提供了
完整的控制SQL。但是，如果不正确的使用，仍然会存在sql 注入的问题。
